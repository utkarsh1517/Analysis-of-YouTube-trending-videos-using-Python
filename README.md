# Analysis-of-YouTube-trending-videos-using-Python
# YouTube Trending Videos Analysis

#Project Overview
This project performs Exploratory Data Analysis (EDA) on YouTube Trending Videos dataset to understand:
- Trending video patterns
- Engagement trends (views, likes, comments)
- Category-wise performance of videos

#Dataset
The dataset is provided in zipped format inside the data/ folder:
data/youtube_trending.zip

#Tools & Technologies
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Jupyter Notebook  

#Project Structure
- data/: Contains the YouTube trending dataset (zipped)
- notebooks/EDA/: Contains Jupyter notebooks for data cleaning and exploratory analysis

#Key Insights 
- Entertainment and Music categories appear most frequently in trending videos.
- Videos with higher views generally receive higher likes and comments.
- User engagement plays an important role in making a video trending.
